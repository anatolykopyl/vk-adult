# VK Adult

Возвращает поиск без ограничений по видеозаписям Вконтакте

![](static/screenshot-1.png)

Недавно ВК убрал из настроек фильтрации данный пункт, но в API возможность передавать соответствующий параметр осталась. Это расширение добавляет пункт обратно в интерфейс и использует [`chrome.debugger`](https://developer.chrome.com/docs/extensions/reference/api/debugger) API для перехвата и изменения запросов.
